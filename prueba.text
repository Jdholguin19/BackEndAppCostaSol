[Rol y contexto]  
Eres un desarrollador senior en PHP, HTML, CSS y JavaScript trabajando en un sistema de chat llamado `ctg_detalle.php`. Este archivo muestra un chat entre un cliente y un responsable. Actualmente, al entrar a la página, el scroll se posiciona al inicio, obligando al usuario a bajar manualmente hasta el final para ver los últimos mensajes.

[Objetivo]  
Implementar una mejora en la navegación del chat para que el usuario pueda ir fácilmente al final de los mensajes, eligiendo una de estas dos opciones:  

1. **Botón/flechita hacia abajo** que, al hacer clic, mueva el scroll al final del chat.  
o
2. **Auto-scroll al final del chat solo al cargar la página por primera vez** (cuando el usuario entra al chat).  

[Requisitos funcionales]  
- La opción de auto-scroll solo debe ejecutarse cuando el usuario **entra** al chat, no cada vez que se refresquen mensajes automáticamente.  
- Si se elige la flechita, debe estar fija (por ejemplo, en la esquina inferior derecha) y ser visible solo si hay mensajes previos que requieran desplazamiento.  
- El scroll debe ir al final **suavemente** (scroll animado).  
- No modificar el backend, solo trabajar en el frontend (HTML/JS/CSS) de `ctg_detalle.php`.  

[Alcance y límites]  
- Mantener el diseño existente del chat.  
- Evitar que la funcionalidad interfiera con la lectura de mensajes anteriores.  

[Formato de salida]  
- Proporcionar el código HTML y JavaScript necesario para ambas opciones, con instrucciones de integración.  

[Estilo y tono]  
Código limpio, comentado y fácil de integrar.
